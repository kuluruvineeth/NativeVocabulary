# A Simple Native to English Words App

### Topics covered while building this app are :
- Multiple Screen Activities
- Fragments
- FragmentManager
- Explicit Intents
- ListView and ArrayAdapter, CustomAdapter
- ViewPager
- TabLayout
- MediaPlayer
- AudioManager
- Recycling Views
